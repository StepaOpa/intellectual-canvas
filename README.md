# intellectual-canvas

## Гайд по настройке venv


### Шаги
#### 1. Создайте виртуальное окружение:
```
python -m venv .venv

source .venv/bin/activate
```
#### 2. Установите зависимости:
```
pip install -r requirements.txt
```


### всегда обновляйте requirements.txt:
```pip freeze > requirements.txt```
